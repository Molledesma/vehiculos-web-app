# vehiculos-web-app
# Análisis de Datos de Vehículos Usados

Esta aplicación web permite visualizar y analizar datos de vehículos usados mediante gráficos interactivos.

## Funcionalidades
- Histograma de distribución del odómetro
- Gráfico de dispersión 3D (odómetro vs año del modelo vs precio)

## Instalación
```bash
pip install streamlit pandas plotly-express


## ¿Cómo ejecutarla?

1. Clona este repositorio
2. Navega a la carpeta del proyecto: `cd nombre_del_repositorio`
3. Instala las dependencias: `pip install -r requirements.txt`
4. Ejecuta la aplicación: `streamlit run app.py`
5. Abre tu navegador en: http://localhost:8501
